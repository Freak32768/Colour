# Introduction

-   Colour(コロールと読む)はその名の通り"色"をテーマにした
    3DパズルTPSゲームです。(パズルかどうかは微妙です。)
-   ColourはGauche(Scheme処理系), Gauche-gl(GaucheのOpenGL拡張)を必要とします。


<a id="orga01c2db"></a>

# System

-   ADで左右に方向回転、WSで移動です。
-   Vで弾を選択し、Bで弾を発射します。
