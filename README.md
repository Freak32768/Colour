# Introduction

Colour(コロールと読む)はその名の通り"色"をテーマにした
3DパズルTPSゲームです。(パズルかどうかは微妙です。)


<a id="org9fcb3e5"></a>

# System

-   ADで左右に方向回転、WSで移動です。
-   Vで弾を選択し、Bで弾を発射します。
