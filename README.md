# Introduction

Colour(コロールと読む)はその名の通り"色"をテーマにした
3DパズルTPSゲームです。(パズルかどうかは微妙です。)
ColourはGNU GPL v3 or laterで配布されています。

# System

-   ADで左右に方向回転、WSで移動です。
-   Vで弾を選択し、Bで弾を発射します。
